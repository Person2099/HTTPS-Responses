# Security Policy

## Supported Versions

| Version           | Supported          |
| ----------------- | ------------------ |
| 1.0 RELEASE       | :white_check_mark: |
| 1.0 (9) BETA      | :white_check_mark: |
| <= 1.0 (8) BETA   | :x:                |

## Reporting a Vulnerability
To report a vulnerability please use the 'report an issue' form in the app's settings
